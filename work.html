Pokédex

A small, responsive web application built on HTML, CSS, vanilla Javascript, jQuery, and Bootstrap. This application loads data from the an external (Pokémon) API, allowing users to retrieve specific data pertaining to each Pokémon, including their height and image. All users have to do is click the button with the name of the pokemon they wish to research and a modal will be opened that reveals these details.

myfavfilmz

A Single Page Application built on the MERN stack, using Node.js and Express to build the API, MongoDB to build the database, and the React.js library for the User Interface. The application exists to allow movie fans the ability to register for an account and create a list of their favorite movies. Users are also able to find out more about these favorite movies, such as details about a movie's genre, director and score on Rotten Tomatoes.

See Hosted Site See Project Repository on Github

Meet App

Meet App is a serverless, Progressive Web Application built with React using Test-Driven Development, using AWS Lambda serverless functions, OAuth2 authentication flow, and the Google Calendar API. It provides users with a lists of events in different cities, as well as useful details about those events. Users are able to filter events by location (along with Recharts data visualization showing number of events in relevant cities), determine the number of events that they with to see on their page, and add a shortcut to their phone for mobile, offline use.
