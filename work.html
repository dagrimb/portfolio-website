My Work


Pokédex

A small, responsive application that loads data from an external (Pokemon) API, allowing users to retrieve specific information pertaining to each Pokemon, including their height and image. All users have to do is click the button with the name of the pokemon they wish to research and a modal will be opened that reveals these details.

    Technologies Used:
  • HTML
  • CSS
  • vanilla JavaScript
  • jQuery
  • Bootstrap
  • The Pokemon API

myfavfilmz-API

A REST API that serves the client-side of the project by connecting it to an externally hosted database. This allows movie fans to, among other things, create accounts, log in, view movie information and create a list of their favorites.

    Technologies Used:
  • Node.js
  • Express.js
  • Mongoose
  • Heroku
  • Postman
  • Mongo & MongDB Atlas
See Project Repository on Github

myfavfilmz-React-client

The React version of my Single Page Application’s client that exists to allow movie fans the ability to register for an account and create a list of their favorite movies. Users are also able to find out more about these favorite movies, such as details about a movie's genre, director and score on Rotten Tomatoes.

    Technologies Used:

  • JSX
  • React Bootstrap
  • React Router
  • React Redux
  • Parcel

Meet App

Meet App provides users with a lists of events in different cities, as well as useful details about those events. Users are able to filter events by location, determine the number of events that they with to see on their page, and add a shortcut to their phone for mobile, offline use.

    Technologies Used:
  • React.js
  • Google Calendar API
  • AWS Lambda
  • OAuth2 authentication
  • Jest
  • Cucumber
  • Puppeteer
  • Lighthouse
  • Atatus
  • Recharts
  • Axios
